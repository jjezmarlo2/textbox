Textbox

The ultimate Ruby chatting tool

Download Textbox

You need Ruby installed for this to work.

If you are using a Macbook, Ruby is already installed.
However, if you are using Windows, you can use Ruby Installer,
and if you are using a Linux, check out this link.

Make a folder named 'textbox'. Make sure the folder is shared with whoever you want to chat with.
Right click on this link and save the link as 'textbox.rb' in the textbox folder.
Note: If you can't right click on the link, click on it, press Ctrl+S on PC (on Macs, Cmd+S) and do the same.

Open Textbox

Macbook and Linux: Open terminal.
Type 'cd' + directory path to textbox folder. For example, 'cd dropbox\textbox'.
Type 'ruby textbox.rb' to open program.

PC: Double-click on textbox.rb file to open Ruby Interpreter.

How to Use Textbox

Make a username, and the program will generate you a password.
Make a chat document. It will ask you who to invite --- enter usernames and then type 'done'.
Note: It will automatically include you in the people who can enter the chat.
The program will bring you automatically to the chat document.

To reload chat, type 'r' into the command prompt.
To quit program, type 'quit'.

If you were the creator of the chat, you can type 'manage' when you are inside the chat.
You can add and take away users, and you can delete the conversation if it gets too long.

Note: If you are using Dropbox to share the folder and you see a blue loading circle on the icon,
wait until it goes away, or it may generate conflicted files.

How to Install Updates

First, delete your previous version of Textbox.
Then re-install Textbox --- it will install to the next version.
To see the version of Textbox you have, type 'version' when it asks you if you are new or not.
